# Note Taker App

A React Native mobile application for secure note-taking with categories, built with Expo and TypeScript.

## Features

- **User Authentication**: Register and login with email, password, and username
- **Protected Routing**: Secure navigation based on authentication status
- **Notes Management**: Full CRUD operations for notes
- **Categories**: Fixed categories (work, study, personal) with ability to add/delete
- **Search**: Search notes by matching words
- **Sorting**: Sort notes by date (ascending/descending)
- **Profile Management**: Update credentials and change password
- **Modern UI**: Beautiful gradient design with blue, violet, and purple colors

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on your device:
- Press `i` for iOS simulator
- Press `a` for Android emulator
- Scan QR code with Expo Go app on your physical device
